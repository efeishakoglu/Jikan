Detailed Description

Implements the routines defined in the header file and routines necessary for those in the background. Pointers of USART wrappers are also defined here.
IRQ definitions with custom ISR implementations are also included at the end of the file.

Byte received in the RDR is not handled if the ISR has not been assigned.
If the wrapper is indeed assigned, the received byte is loaded at the next index of the ring buffer and the ISR HIERODULE_USART_Wrapper::RX_Handler is called if it's not NULL.
